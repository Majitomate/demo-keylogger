# demo-keylogger
Demo Educativa de KeyLogger en lenguaje JavaScript

Keylogger Demo con Fines Educativos
Esta es una aplicación web sencilla diseñada para demostrar cómo los navegadores gestionan e interceptan los eventos del teclado utilizando JavaScript puro (Vanilla JS).

Objetivo Didáctico
El propósito de este proyecto es ayudar a comprender:
- Cómo funciona la interfaz KeyboardEvent en el DOM.
- La diferencia entre propiedades como key, code y keyCode.
- Cómo se pueden capturar combinaciones de teclas (modificadores como Ctrl, Shift, Alt).
- La importancia de no introducir datos sensibles en sitios web que no sean de confianza.

Características
- Captura en tiempo real: Visualización instantánea de la tecla presionada.
- Estadísticas de sesión: Conteo de teclas especiales vs. caracteres imprimibles.
- Log de eventos: Registro detallado con marcas de tiempo (timestamp).
- Exportación de datos: Función para descargar el historial de captura en un archivo .txt local.
